The Advisory Process

The general framework for an advisory engagement is:

1 Initial Consultation

Set up a quick call or face-to-face to dig down and understand the problems at hand. Only then can we work towards a plan of attack.

2 Strategic Analysis

Let's take a step back and deliberate on what is holding you back. Is it competition? Is it operations? Is it technology?

3 Implementation Plan

Make it happen. A plan is only as good as it's execution. Let's scope out what needs to be done and when it can be done by.

4 Monitoring Assessment

Measure twice, cut once. Progress must be made. How can you know if you've finished this marathon without knowing where the finish line is?
